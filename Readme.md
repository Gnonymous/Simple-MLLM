# Simple-MLLM

这是一个简易的在本地部署多模态大模型（MLLM）的实例，支持包括图片、文字以及语音（正在更新）多种模态的输入。

如果你对多模态大模型（MLLM）感兴趣但是不知如何上手，来看看这个项目，一个简易的项目让你感受到多模态大模型的魔力👍
